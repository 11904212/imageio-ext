The Image I/O-Ext is a project which aims to extend the Java Sun's ImageI/O framework capabilities. It is an extensibile tool of libraries to mainly support a wide set of data formats

Main Features

The main core module of the project is gdalframework, a framework leveraging on the Geospatial Data Abstraction Library (GDAL) via SWIG's generated JAVA bindings to provide support for a reach set of data formats. Here below there is a brief list of the actually supported formats:


Format Name Data access capabilities*
Arc/Info ASCII Grid R
DTED R
ECW R
Envisat R
GeoTiff R/W
HDF4 R
JPEG R
JPEG 2000 (Kakadu) R/W
JPEG 2000 (MrSID) R
JPEG 2000 (ECW) R
MrSID R

Beside of this, there are some additional main modules:

  • imagereadmt providing a specific ImageRead operation implementation with support for multithreading.
  • customstreams is another main module containing a wide set of extended input/output streams
  • jhdfaccessis a revisited version of the ncsa JHDF libraries which provide access to HDF format. The actuall state of our access library provides read access to the HDF4 format.

Finally, Image I/O-Ext provides a set of additional plugins (which dont rely on GDAL):


Format Name Data access capabilities* Additional Notes
Arc/Info ASCII Grid R/W
JPEG R Built on the JMagick library
JPEG 2000 R Built on the Kakadu library. This plugin version leverages on
the powerful multithreading capabilities provided by
the imagereadmt module.

Data access capabilities*: R=Readonly capabilities; R/W=Read and write capabilities

Feedback and Discussion

Five main Image I/O-Ext mailing lists are available in this project: dev 'at' imageio-ext.dev.java.net, announce 'at' imageio-ext.dev.java.net, issues 'at' imageio-ext.dev.java.net, commits 'at' imageio-ext.dev.java.net, and users 'at' imageio-ext.dev.java.net (replace 'at' with @). Click here to subscribe to any of these lists. The dev list is a list for developers to discuss project development issues/artifacts and requirements. The announce list is a low-volume moderated mailing list for announcing ImageI/O-EXT news and information. The issues list is for automated issues tracking and the commits list is for automated subversion commits messages. Lastly, the users is a general discussion list for the projects end users.
