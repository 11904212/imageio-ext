ImageI/O-EXT Project Home Page

 

ImageI/O-EXT is an extensible tool of libraries extending the Image I/O capabilities to mainly support a wide set of data formats.

The main core module of the project is gdalframework a framework leveraging on GDAL via SWIG's generated JAVA bindings to provide support for a reach set of data formats. 
This work has been started with the Google Summer of Code in 2006 by Daniele Romagnoli mentored by Simone Giannecchini (GeoSolutions).

 

Beside of this, there are two additional main modules:

imagereadmt providing a specific ImageRead operation implementation with support for multithreading.
customstreamsis another main module containing a wide set of extended input/output streams

Finally, several plugins exists to support the following formats: ASCII ArcGrid, GeoTiff, JPEG2000 (Kakadu based), ECW, MrSID, HDF4.

JPEG2000 plugin leverages on the powerful multithreading capabilities provided by the imagereadmt module.
